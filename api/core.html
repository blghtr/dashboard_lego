Abstract base class with data processing pipeline.

Pipeline stages: 1. Load raw data (_load_raw_data) 2. Preprocess (PreProcessor.process) 3. Filter (DataFilter.filter) 4. Cache result at each stage

Each stage can be cached independently based on its relevant params only, enabling efficient updates when only filtering changes.
